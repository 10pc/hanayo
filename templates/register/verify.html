The Guide™, tl;dr version

  1. Download the server switcher
  2. Extract it, disable your antivirus, execute it as administrator
  3. Click on install certificate, then Yes
  4. Make sure the switcher says you're playing on Ripple server, if not click On/Off
  5. If it says something about read-only mode, go to C:\Windows\System32\Drivers\etc, open the hosts' file Properties and make sure it's not set to "Read only".

If you're still having trouble, then again, get on our Discord server's #help channel. To have a better chance to be answered, say "strawberry" anywhere in your messages so that we know you have actually read this.
